Método de pago de Woocommerce usando Visanet

== Description ==

Este plugin permite el procesamiento de pagos con tarjetas Visa a través de Visanet Dominicana.

Tan pronto active el plugin, puede ir a la pantalla de configuración donde encontrará los campos necesarios para empezar a procesar pagos.
